# Todo App

a simple all-in-one app where you can create tasks, group them into folders, edit them and delete entire folders with their respective tasks.

## Technologies

DB

```bash
PostgreSQL 14.1.1
```

Backend

```bash
NestJS 8.0.0
Typeorm 0.2.41
Typescript 4.5.2
```

Frontend

```bash
React 17.0.2
React-icons 4.3.1
CSS 3
```
